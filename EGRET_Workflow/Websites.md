## URL Reference for Obtaining USGS Codes from Websites
--------------------------------------------------------

### USGS EGRET on Github
https://github.com/USGS-R/EGRET/wiki

---------------------------------------------------------

### National Map
http://maps.waterdata.usgs.gov/mapper/index.html

---------------------------------------------------------

### 8-digit Site ID from National Water Information System
http://waterdata.usgs.gov/nwis

---------------------------------------------------------

### 5-digit parameter codes definitions (for real time data)
http://go.usa.gov/bVDz

---------------------------------------------------------

### 5-digit stat code (for continuous data/descriptive stats)
http://nwis.waterdata.usgs.gov/nwis/help/?read_file=stat&format=table

---------------------------------------------------------

### Additional info about sites
http://waterservices.usgs.gov/rest/Site-Test-Tool.html

---------------------------------------------------------

### Additional info about parameters and parameter codes
http://nwis.waterdata.usgs.gov/nwis/pmcodes/

---------------------------------------------------------

### Info about publication remarks - "A"=approved for publication  and "P"=provisional
http://waterdata.usgs.gov/usa/nwis/help?codes_help

---------------------------------------------------------

### Daily data are pulled from this REST Service
http://waterservices.usgs.gov/rest/DV-Test-Tool.html

---------------------------------------------------------

### Instantaneous values are pulled from this REST Service
http://waterservices.usgs.gov/rest/IV-Test-Tool.html

---------------------------------------------------------

### WATER QUALITY data portal
http://www.waterqualitydata.us/

### Storet Water Quality Data (USEPA)
http://www.epa.gov/storet/dw_home.html

---------------------------------------------------------
