## URL Reference for Obtaining USGS Codes from Websites
--------------------------------------------------------

### USGS EGRET on Github
https://github.com/USGS-R/EGRET/wiki

---------------------------------------------------------

### 8-digit Site ID from National Water Information System: Mapper http://maps
http://waterdata.usgs.gov/nwis

---------------------------------------------------------

### 5-digit parameter codes definitions (for real time data)
http://go.usa.gov/bVDz

---------------------------------------------------------

http://nwis.waterdata.usgs.gov/nwis/help/?read_file=stat&format=table

http://waterservices.usgs.gov/rest/Site-Test-Tool.html

http://nwis.waterdata.usgs.gov/nwis/pmcodes/
